# Multi-Factor Authentication (MFA)

## What is MFA?

Multi-Factor Authentication (MFA) is a security method that requires users to provide two or more verification factors before accessing an account or resource.

## Why is MFA Important?

MFA provides an additional layer of security and helps protect user accounts from unauthorized access.

## Common Authentication Factors

- Something you know – Password or PIN
- Something you have – Mobile phone or security key
- Something you are – Fingerprint or Face ID

## Microsoft Entra ID MFA

Microsoft Entra ID supports MFA to help protect identities and resources.

## Key Benefits

- Protects against stolen passwords
- Reduces unauthorized access
- Adds an additional security layer
- Supports Zero Trust security principles
