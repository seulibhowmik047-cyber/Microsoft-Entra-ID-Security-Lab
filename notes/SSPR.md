# Self-Service Password Reset (SSPR)

## What is SSPR?

Self-Service Password Reset (SSPR) allows users to reset or change their passwords without requiring help from an administrator or help desk.

## Why is SSPR Important?

SSPR helps reduce password-related support requests and improves user productivity.

## Key Benefits

- Reduces help desk workload
- Allows users to reset passwords
- Improves user experience
- Supports identity security

## Authentication Methods

Depending on the configuration, users may verify their identity using supported authentication methods.
