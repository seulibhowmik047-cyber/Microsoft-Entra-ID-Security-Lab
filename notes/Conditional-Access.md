# Conditional Access

## What is Conditional Access?

Conditional Access is an identity security capability in Microsoft Entra ID that helps organizations control access based on specific conditions.

## Common Conditions

- User or Group
- Location
- Device
- Application
- Risk
- Sign-in Information

## Common Access Controls

- Grant Access
- Block Access
- Require MFA
- Require Compliant Device

## Why is Conditional Access Important?

Conditional Access helps organizations implement Zero Trust security principles and protect cloud resources.
